При анализе использовалась утилита jhsdb jmap --heap --pid
и графическая jconsole
В входе анализа программа работает стабильно, потребление памяти примерно ~200мб
все работает в одном потоке.
При уменьшение памяти выделеное JVM c флагами -Xms128m -Xmx128m
программа падает с OutOfMemmoryError.
При увеличении памяти до -Xms256m -Xmx256m программа работает стабильно.
